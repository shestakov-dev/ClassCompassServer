# ClassCompass Server

NestJS backend for the ClassCompass platform.

## Local start

Copy over the env example to `.env` and edit the values where needed.

Apply migrations (local dev):

```bash
pnpx prisma migrate dev
```

```bash
pnpm install

pnpm run start:dev
```
